Aplicação Exemplo

Esta aplicação é um exemplo de como utilizar a biblioteca Javascript do painel Dife em um site para torná-lo gerenciável. Os conteúdos desta página veem de nosso painel.

Acessar Dife Ver no GitHub

Projetos Realizados

Nesta seção os projetos são apenas de exemplo. A lista e dados do projeto são buscados via Javascript e veem do painel Dife.

Nosso Time

O nosso time também é apenas exemplos. Os dados também são listados diretamente do painel de administração, onde poderão ser gerenciados.

Fale Conosco

O formulário de contato é feito localmente, no layout, mas após preenchê-lo e clicar em enviar, o Javascript envia ao painel de adminsitração. O seu cliente irá receberá um notificação por e-mail.
